# Video Scripts — Production Notes

## Overview

5 demo scripts for Flowtech Advisory's digital product suite. Each follows a consistent Hook/Demo/CTA structure targeting construction professionals.

## Structure (all scripts)

- **Hook (5s):** Problem statement — relatable pain point
- **Demo (40-50s):** Product solving the problem with specific feature callouts
- **CTA (10-15s):** Price, Gumroad link, Flowtech branding

## Recording Guidelines

### Screen Recordings
- **Resolution:** 1920x1080 minimum (4K preferred for zoom-ins)
- **Browser/App:** Use Edge or Chrome for Power Automate portal; Excel desktop for spreadsheets
- **Mouse movement:** Slow, deliberate — no rapid scrolling. Pause on key features for 2-3 seconds
- **Zoom:** Use screen zoom (Ctrl+scroll) to highlight specific cells, flow steps, or PDF sections
- **Clean desktop:** Hide bookmarks bar, close unrelated tabs, use a neutral wallpaper

### Voiceover
- **Tone:** Confident, practical, no hype. Speak as a peer, not a salesperson
- **Pace:** ~150 words per minute. Leave 1-second pauses between sections
- **Avoid:** "Amazing", "incredible", "game-changer" — let the product speak for itself

### Editing
- **Transitions:** Simple cuts or brief crossfades. No flashy transitions
- **Text overlays:** Product name + price at start of CTA. Feature callouts as lower-thirds during demo
- **Music:** Subtle background track, low volume. Instrumental only
- **Thumbnail:** Product screenshot with bold text overlay of the main value prop

### Distribution
- **Primary:** Gumroad product pages (embed or link)
- **Secondary:** YouTube (with timestamps in description), LinkedIn, Twitter/X
- **Aspect ratios:** 16:9 for YouTube/Gumroad, consider 9:16 vertical cuts for social

## File Naming Convention

Final rendered videos: `flowtech-demo-{product-slug}-v{version}.mp4`

Example: `flowtech-demo-template-pack-v1.mp4`
